Tipos de Selectores CSS

1. Selector Universal

Selecciona todos los elementos de la página.

* { ... }

2. Selector por Elemento

Selecciona elementos HTML específicos.

body { ... }

3. Selector por Clase

Selecciona elementos con una clase específica.

.header-principal { ... }

4. Selector por ID

Selecciona un elemento con un ID específico.

#descripcion { ... }

5. Selector Descendiente

Selecciona elementos anidados dentro de otro elemento.

.menu-navegacion a { ... }

6. Selector por Lista

Selecciona elementos de lista específicos.

.lista-hobbies { ... }

7. Artículos Descendientes de un Elemento con Clase

Selecciona artículos dentro de elementos con una clase específica.

.seccion-hobby article { ... }

8. Hijos Directos

Selecciona elementos que son hijos directos de otro elemento.

.contenido-principal > section { ... }

9. Hermanos Adyacentes

Selecciona elementos que son hermanos inmediatos de otro elemento.

h2 + article { ... }

10. Hermanos Generales

Selecciona elementos que son hermanos de otro elemento.

h2 ~ p { ... }

11. Pseudoclase

Selecciona elementos en un estado específico.

.enlace-red:hover { ... }

12. Pseudoelemento

Selecciona una parte específica de un elemento.

h2::after { ... }
